Automotive ML Algorithm Advisor

An intelligent, interactive ML system that automatically analyzes any dataset and recommends the most suitable machine learning model — with explainability, performance insights, and smart safeguards.

Automotive ML Algorithm Advisor is a Streamlit-based application that helps users:

Upload any tabular dataset (automotive, EV, sales, sensor, etc.)
Automatically detect whether the problem is Classification or Regression
Benchmark multiple ML models
Recommend the best-performing model
Explain why that model fits the data
Support manual model selection & comparison
Handle large datasets, imbalanced data, and time-series data intelligently

This tool is designed for:

Data scientists
ML beginners
Automotive analytics teams
Hackathon & demo use-cases

Installation & Local Run

# Clone repository
git clone https://github.com/<your-username>/automotive-ml-advisor.git
cd automotive-ml-advisor

# Create virtual environment (optional)
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
